Infant Microbiome Relative Abundance Viewer

Overview
This Shiny app allows users to explore infant gut microbiome composition across different dietary groups in the first year of life. Users can filter data by diet, age, and gender to visualize relative abundance of bacterial genera.

Features

Filter microbiome data by infant diet, age in months, and gender
View relative abundance of bacterial genera as stacked bar charts
Compare microbiome composition between male and female infants side by side


Data Source
Data was obtained from the Borenstein Lab curated microbiome-metabolome dataset repository:
[HE_INFANTS_MFGM_2019](https://github.com/borenstein-lab/microbiome-metabolome-curated-data/tree/main/data/processed_data/HE_INFANTS_MFGM_2019)

Original Study
He, X., Parenti, M., Grip, T., Lönnerdal, B., Timby, N., Domellöf, M., Hernell, O., & Slupsky, C. M. (2019). Fecal microbiome and metabolome of infants fed bovine MFGM supplemented formula or standard formula with breast-fed infants as reference: a randomized controlled trial. Scientific Reports, 9(1), 1-14. https://doi.org/10.1038/s41598-019-47953-4
